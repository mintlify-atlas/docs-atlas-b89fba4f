## What you'll build

The console uses an **ESP32-C3** microcontroller, a **1.3-inch SH1106 OLED display**, and a **single analog 5-key button module** — just three components connected with jumper wires.

<Steps>
  <Step title="Gather the hardware">
    You need an ESP32-C3 dev board, a 1.3-inch SH1106 OLED, and an analog 5-key button module. See the [components guide](/hardware/components) for exact parts.
  </Step>
  <Step title="Wire everything up">
    Connect the OLED via I2C (SDA → GPIO 8, SCL → GPIO 9) and the button module to GPIO 0. Full wiring details in the [wiring guide](/hardware/wiring).
  </Step>
  <Step title="Flash the firmware">
    Install Arduino IDE, add the ESP32-C3 board support package, install the U8g2 library, and upload `CMF_GAME.ino`. Step-by-step in the [firmware setup guide](/firmware/setup).
  </Step>
  <Step title="Play">
    Use the analog button to navigate the menu and launch any of the four built-in games. See [controls](/games/controls) for the button mappings.
  </Step>
</Steps>

## Key features

- **Four playable games** — Snake, Pong, Dodge, and a DOOM-style micro FPS (Battleship in progress)
- **Raycasting renderer** — a DOOM-inspired 3D perspective on a 128×64 pixel display
- **Minimal hardware** — three components, no PCB required
- **Single-file firmware** — all game logic in one readable `CMF_GAME.ino` file
- **MIT licensed** — free to modify, extend, and share

<Note>
  This project is designed for makers and students learning embedded systems. The firmware is intentionally simple and well-commented to serve as a learning resource.
</Note>

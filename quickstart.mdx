---
title: Quickstart
description: Wire up the hardware, install dependencies, and get the DIY Gaming Console running in six steps.
---

This guide walks you from an empty workbench to a working console. The whole process takes about 20–30 minutes.

<Steps>
  <Step title="Gather the hardware">
    Make sure you have all five components before you start:

    | Component | Notes |
    |-----------|-------|
    | ESP32-C3 Development Board | Any standard ESP32-C3 DevKit |
    | 1.3-inch OLED Display (SH1106, 128×64) | I2C interface |
    | Analog 5-Key Button Module | Single-wire analog output |
    | USB-C Cable | For programming and power |
    | Jumper Wires | Male-to-female for breadboard connections |

    <Note>
      The firmware defaults to the **SH1106** driver. If you have an SSD1306 display instead, you will swap one line in Step 5 — both displays are fully supported.
    </Note>
  </Step>

  <Step title="Install Arduino IDE">
    1. Download and install the latest **Arduino IDE** from [arduino.cc](https://www.arduino.cc/en/software).
    2. Open the IDE, go to **Tools → Board → Boards Manager**.
    3. Search for `ESP32` and install the **ESP32 by Espressif Systems** package.
    4. Once installed, go to **Tools → Board** and select **ESP32C3 Dev Module** (or the closest match for your ESP32-C3 board).
  </Step>

  <Step title="Install the U8g2 library">
    The firmware uses U8g2 for all OLED rendering.

    1. In Arduino IDE, go to **Tools → Manage Libraries**.
    2. Search for `U8g2`.
    3. Install **U8g2 by oliver**.

    <Warning>
      Do not install "U8glib" — that is an older, incompatible library. Make sure the author shown is **oliver**.
    </Warning>
  </Step>

  <Step title="Wire the hardware">
    Connect the three components to the ESP32-C3 using jumper wires:

    | Component | Pin | ESP32-C3 GPIO |
    |-----------|-----|---------------|
    | OLED | SDA | GPIO 8 |
    | OLED | SCL | GPIO 9 |
    | OLED | VCC | 3.3V |
    | OLED | GND | GND |
    | Button Module | SIG | GPIO 0 (ADC) |
    | Button Module | VCC | 3.3V |
    | Button Module | GND | GND |

    For a full schematic and wiring diagram, see [Hardware & Wiring](/hardware/wiring).
  </Step>

  <Step title="Open and configure the firmware">
    Open `code/CMF_GAME.ino` in Arduino IDE.

    At the top of the file, the OLED driver is configured here:

    ```cpp
    U8G2_SH1106_128X64_NONAME_F_HW_I2C u8g2(U8G2_R0, U8X8_PIN_NONE);
    // If SSD1306, use below instead
    // U8G2_SSD1306_128X64_NONAME_F_HW_I2C u8g2(U8G2_R0, U8X8_PIN_NONE);
    ```

    - **SH1106 display (default):** no changes needed — the first line is already active.
    - **SSD1306 display:** comment out the first line and uncomment the second line.

    No other configuration is required for a standard build.
  </Step>

  <Step title="Upload to the board">
    1. Connect your ESP32-C3 to your computer via the USB-C cable.
    2. In Arduino IDE, go to **Tools → Port** and select the COM port for your board.
    3. Click the **Upload** button (right-arrow icon).
    4. Wait for the compile and upload to complete.

    Once uploaded, the console boots with a pulsing dot animation, then displays the **MINI GAMES** title screen and the game selection menu.

    <Tip>
      Press **SW5** (the RESET button) from within any game to exit back to the main menu.
    </Tip>
  </Step>
</Steps>
